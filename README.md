# AI Resume Fit Analyzer

An AI-powered web application that analyzes resume compatibility with job descriptions using NLP techniques.

## Features
- Semantic similarity matching
- ATS compatibility check
- Skill gap analysis
- Interactive dashboard

## Tech Stack
Python, Streamlit, Sentence Transformers, PyMuPDF, Plotly

## How to Run
pip install -r requirements.txt
streamlit run app.py
